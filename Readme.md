# Front accessible

## Features
*   React 16
*   Webpack 4
*   Styled Components 4
*   Babel 7
*   Styleguidist 7
*   Hot Module Replacement

## Installation:
### Development Mode
*   npm start:dev
*   visit `http://localhost:8080`

### Production

## Style Guide
*   To start development Server : `npx styleguidist server`
*   visit `http://localhost:6060`
*   Build command : `npx styleguidist build`
